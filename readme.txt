Include SDL2 files in order to run. The required files are in the folder SDL2 Files. Run the game in debug x86 mode or it won't work.
Add SDL2, SDl2 image, SDL2 ttf, and SDL2 mixer into VC+ directory on visual studios. Do the equivalent for any other IDEs.
